Я проектирую и внедряю production-ready AI-системы для реальных бизнес-задач. Мой подход — это 19 лет практического опыта в IT: от управления службой ИТ из 40 человек в одном из крупнейших морских пароходств России до построения RAG-пайплайнов, мульти-агентных архитектур и корпоративных DWH для современных клиентов.

Я не просто "вайбкожу с промптами". Я проектирую системы, пишу код, разворачиваю в продакшне и отвечаю за результат.

### 💼 Чем я занимаюсь

- 🏗️ **AI-архитектурный консалтинг** — аудит бизнес-процессов, подбор оптимального LLM-стека, конкретная архитектурная схема с обоснованием каждого решения и дорожной картой внедрения
- 🤖 **Мульти-агентные системы** — иерархические архитектуры (оркестратор + исследователь + разработчик + дизайнер) с domain routing, протоколом Shared State и multi-provider LLM fallback через OpenRouter
- 🔍 **RAG-пайплайны** — мультимодальные retrieval-системы (текст, vision, аудио/STT) с FAISS / ChromaDB vector search
- 📊 **Корпоративные DWH** — PostgreSQL с SQLAlchemy 2.0, миграции Alembic, оркестрация ETL через n8n
- ⚙️ **Автоматизация бизнеса** — объединение разрозненных SaaS / PMS / CRM в единый AI-пайплайн
